Dictionaries (Explanation) #

To understand about dictionaries, we have tp understand about its components namely:

1. Keys: So keys are the a kind of unique id assigned it a value. Such that we can access the values with respect to its key. Keys for a value can't be changed though it's value can be reassigned with new key. Note- List can't be a key.

2. Values: Similarly, values are what assigned to every keys and are fetched by mentioning its key.

Let's examine a case of assigning dictionary:

dict = {'a':1, 'b':2, 'c':3}

Here, a, b, c are the keys and 1, 2, 3 are the respective values assigned to them.

Accessing Values in Dictionary #

Now as we discussed earlier, we can access the values of dictionaries by using its respective key.

dict = {'a':1, 'b':2, 'c':3}
print(dict['a'])

Here what we did is, declared a dictionary with keys and values and used a key to fetch its value. Here, key a whose value is 1 which will be the output after passing the key.

Updating Dictionary #

Dictionaries are one of the mutable datatypes. Hence, updating is a thing. Updating can be done in many ways like adding new set of keys and there respective values or by changing the value of existing ones. 

dict = {'a':1, 'b':2, 'c':3}
dict['d'] = 4

Here, we are adding new set key and value which is 4 assigned to key 'd'.

dict['a'] = 0

Similarly, we can modify the existing set of values. Here, we assigned a new value i.e, 0 to key 'a'.

Delete Dictionary Elements #

Now, referring dictionary as mutable datatype means we can delete elements too. It is simple yet necessary feature. There are mainly three options available, deleting a specific element, deleting all entries or deleting the whole dictionary. 

del dict['a']

Here, this code just deletes the element whose key is mentioned. This is used when we need to delete some elements from the dictionary. 

dict.clear()

Here, this helps us to clear the whole dictionary i.e., it deletes all elements from the dictionary making it empty.

del dict

This straight deletes the whole dictionary as if it was never declared. Note: It deletes the variable of dictionary too.
