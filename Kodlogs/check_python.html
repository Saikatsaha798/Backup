Searching for Python IDLE #

Now, starting with a beginner friendly way. For that, we will be checking if IDLE is installed.

FAQ An integrated development environment (IDE) for Python is called IDLE (Integrated Development and Learning Environment).

This IDLE is installed when you installed Python in your PC. This is basically a simple and light editor for python programming often recommended to beginners due to simple interface and no auto correction.

First solution, to find and check if its is present. That we do by searching it in cortana or start - search bar. This will give you the options and if it is present, Voila! it is Present in PC.

Caution Don't think Python is not installed right away if IDLE not found. Python named shell might be present which will also do.

Checking Python in Environment Variables #

Now as we have seen the easier way, it can be also checked by checking Python in Environment Variables. This way tells you whether the Python is properly installed or not.

To work with Python, you will need to call Python interpreterin different terminal, powershell or cmd.

For this, just open Command Prompt or Powershell and write "python" and ENTER. If Python shell opens in the same window then, it's clear Python is properly installed. Else if, 'python' not found comes up its not properly installed.

Note Sometimes, Python is installed but the Path is not present in Environment. But, in many cases you need to call the interpreter in different places, hence as mentioned its not properly installed.

Tip Make sure to tick the option in ADD TO PATH in advanced option in Python Installer while Installing to make sure Python path gets added to Environment Variables.

FAQ If you know python interpreter is installed, yet not present in Environment Variables. Then just search for the Python directory or just open file location to get the folder location. Now, just simply copy the folder location and then search Environment Variables and there it will have option "Edit the system environment variables" and there just click option "Environment Variables" and there add the folder location copied in the Path option and paste there. Now, just restart the PC and just enjoy.
