Searching (Explanation) #

Now, starting with the term "searching", it is the process of locating a specific element present in a set of elements that is the process of searching. A table, a file, or a record could make up the element. 

Now, there are a few steps in searching :

Firstly, we just check whether the element of search is present or not by returning only True or False. 

Secondly, we check the elements present in the list and take note of the indices. In this tutorial, we will be going over this step.

Tip The first one is the beginner's one. It is advisable to visit earlier mentioned tutorial if you are a beginner.

Solution #

Now that you understand the task, let's dive into the solution. Basically, the algorithm is the same and easy, but will show two ways to do the same. What we will do is traverse the list and check one by one if it matches the item we are searching for. We simply put it in the list of occurrences.

Directly traversing using Range function#

def search_range(l, a):
    occurences = []
    for i in range(len(l)):
        if a == l[i]:
            occurences.append(i)
    return occurences

Given the code, we can see we are taking parameters, alist named l and an  item as a and returning all occurences of the element as a list named occurences.

Now let's discuss it in detail:
Firstly, occurences is the list created, then we run a loop to iterate over the length of the list using range and len function. Inside the loop we have put an  If statement which checks if the iterating element of the element is the item we are searching for or not. And if the element and the item we are searching for are the same, it appends the index in the list called "occurences" using the append function.

Caution If the list you are dealing with is way too big, then using it with the above algorithm will take a long time and might even crash, so numpy is recommended for such use.

Using enumerate function #

def search_enumerate(l, a):
    occurences = []
    for i in enumerate(l):
        if a == i[-1]:
            occurences.append(i[0])
    return occurences

Here we are using the Enumerate function, to do the job with the same parameters, a list named as l and an item as a and similarly, returning all occurences of the element as a list named occurences.

Now, first understand enumerate a bit,
An iterable is given a counter by the enumerate() method, which returns the object as an enumerating object. Then, using the list() method, this enumerated object can be turned into a list of tuples for use in loops. So basically, we input a list and get tuples of items with their indices. 

Now, though the algorithm is similar, let's discuss it in detail:
Firstly, the list is created, then we run a loop to iterate over the length of the list by getting tuples of elements and their indices using the enumerate function. Inside the loop we have put an If statement which checks if the element present in the last part of the tuple that is "-1" index s the item we are searching for or not. And if the element and the item we are searching for are the same, then append the index that is again present in the first part of the tuple, i.e., using the "0" index in the list called occurences using append function.

FAQ "Traverse" is a word used often here. A data structure operation is traversing. It is a process that allows you to access each and every element of any data structure, including arrays, linked lists, and other types of data structures. Accessing means making at least one trip to each element in order to display them to the user or carry out an operation on each element.
