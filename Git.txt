**Git Tutorial**

git status 								// Current status of the Repository.
git add .  								// Adds all file to the staging area.
git commit -m "Version NO" 		// Saves repository corresponding to Version name or 													message.
git config --global user.name 	// Changes the username of author in all scopes.
git config --global user.email 	// Changes the email of author in all scopes.

*connecting to Github*

1. Create a SSH key
2. Add it to Github
3. git remote add origin $SSH Link to Repo$

git push -u origin master      	// Push to Github repository
